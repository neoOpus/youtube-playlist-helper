# User Testing Protocol: YouTube Playlist Helper (Pro Edition)

## 1. Test Overview
*   **Methodology:** Remote, Unmoderated "Think Aloud" test.
*   **Participants:** 5 regular YouTube users (varying technical proficiency).
*   **Environment:** Production-ready Chrome Extension Dashboard.

## 2. Test Objectives
1.  Verify the discoverability of the "Manage Hub" and "Global Sync" features.
2.  Assess the intuitiveness of the new "Luminous Hover" selection pattern.
3.  Evaluate the effectiveness of the 10-second "Undo" window for deletions.
4.  Measure general system delight and "Professional" brand perception.

## 3. Scenarios & Tasks
| Task # | Scenario | Success Metric |
| :--- | :--- | :--- |
| **Task 1** | You want to change the UI to the "Pro Red" theme. Find the setting and apply it. | Successful theme change in < 15s. |
| **Task 2** | You accidentally deleted a critical playlist. Use the Undo system to restore it. | Successful restoration via Toast. |
| **Task 3** | Create a new playlist and add 5 videos using the "Bulk Intake" tool. | Successful creation without errors. |
| **Task 4** | Navigate to the "Manage Hub" and find the "System Health" audit. | Successful discovery of Auditor. |

## 4. Post-Test Survey (SUS)
*Rate on a scale of 1 (Strongly Disagree) to 5 (Strongly Agree):*
1.  I found the system easy to use.
2.  The visual design feels modern and professional.
3.  I felt confident navigating the dashboard.
4.  The animations and interactions were helpful, not distracting.
