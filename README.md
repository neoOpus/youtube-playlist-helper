# YouTube Playlist Helper

A clean, professional extension for managing and organizing YouTube playlists.

## Features
- **Clean UI:** A focused, high-contrast interface for distraction-free management.
- **De-duplication:** Easily find and remove duplicate videos.
- **Cloud Sync:** Synchronize your collections across devices.
- **Privacy First:** Local-first storage with optional encrypted cloud sync.

## Technical Stack
- **Svelte 5** (Runes)
- **TypeScript**
- **Vite**
- **Manifest V3**

## Installation & Build
1. Install Node.js and npm.
2. Run `npm install` from the root.
3. Run `npm run build` to build the extension.
4. Load the `projects/extension` directory as an unpacked extension in Chrome.
