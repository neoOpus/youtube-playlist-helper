# Form Recovery Suite - Archive & Research

This directory contains several projects related to browser form recovery. Each sub-project is self-contained and ready to be moved to its own repository.

## Projects Included:

- **Project-Typio-Legacy**: Original Typio Form Recovery source and its community variants.
- **Project-Lazarus-Archive**: Archive of the Lazarus Form Recovery addon research.
- **Project-Form-Recovery-Tools**: Shared scripts, userscripts, and utilities for evaluating form identification robustness.
- **Project-Modernization-Research**: Documentation and vision for a "SOTA" 2026 Form Recovery tool (v2).
