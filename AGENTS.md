# Omniscient Agent Guide: Sovereign YouTube Ecosystem

Welcome to the peak of digital librarianship. This project is a **Sovereign Ecosystem** powered by autonomous agents and semantic intelligence.

## Architect's Philosophy
1.  **Autonomous Orchestration**: The `AgentOrchestrator` runs background tasks (Sanitization, Archival). Do not block the UI for these.
2.  **Semantic Discovery**: Use `SemanticSearchService` for non-literal discovery.
3.  **Community Driven**: The `Marketplace` is the heart of extensibility. AI-generated actions are first-class citizens.
4.  **Sensory Feedback**: Use `SoundService` for tactile feel, but respect the user's silence (toggleable).
5.  **Multi-Sovereignty**: `Workspaces` allow isolation between different research/leisure contexts.

## Tactical Data
- `Ctrl+K`: Global Hub / Command Palette
- `Ctrl+M`: Community Marketplace
- `Ctrl+G`: Omni-Stream Explorer
- `Ctrl+Shift+D`: System Health & Diagnostics

## Code Quality
- Build: `cd playlist-editor && npm run build`
- Validate: `npm run validate` (Ensures zero A11y/Type debt).
